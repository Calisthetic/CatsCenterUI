# CatsCenter UI

## Setup project

Create react app

```bash
npx create-react-app my-app
```

Install Tailwind

```bash
npm install -D tailwindcss
```

Add tailwind config to project

```bash
npx tailwindcss init
```

Add React Router

```bash
npm i -D react-router-dom
```
